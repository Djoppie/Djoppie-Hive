# HR Personeelsbeheer - Gemeente Diepenbeek

React/TypeScript webapplicatie voor HR-personeelsbeheer van het lokale bestuur Diepenbeek.

## Functionaliteiten

- **Dashboard** - Overzicht van alle statistieken (medewerkers, validatie, sectoren)
- **Personeelslijst** - Volledig overzicht met zoeken, filteren, sorteren en CSV-export
- **Validatie** - Workflow voor diensthoofden en sectormanagers om gegevens te controleren
- **Uitnodigingen** - Aanmaken en versturen van uitnodigingen (personeelsfeest, communicatie)
- **Rollen & Rechten** - Beheer van profielen binnen de organisatie
- **Azure AD Import** - Importeer gebruikers uit Azure AD/Entra of CSV

## Opstarten

```bash
cd hr-personeel
npm install
npm run dev
```

## Tech Stack

- React 19 + TypeScript
- Vite
- React Router
- Lucide React (icons)
