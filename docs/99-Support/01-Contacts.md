## üêõ Reporting Bugs

When you find a bug, please report it with:

1. **Title**: Concise description of the problem
2. **Severity**:
   - Critical: System unusable
   - High: Major feature broken
   - Medium: Feature partially broken
   - Low: Minor issue/UI glitch

3. **Steps to Reproduce**:
   - Exact steps to recreate the issue
   - Include any filters/searches used
   - Include any data specifics

4. **Expected Behavior**: What should happen

5. **Actual Behavior**: What actually happens

6. **Screenshots/Video**: Visual proof if possible

7. **Environment**:
   - Browser (Chrome, Edge, Firefox, Safari)
   - Browser version
   - Operating System (Windows, Mac, Linux)

**Send to**: ict@diepenbeek.be with subject `[DJOPPIE-HIVE] [BUG]`
